Problem solving is writing an original program that performs a particular set of tasks and meets all stated constraints.

I was blocked on a simple problem

When I was writting code during foundation sprint four I was blocked on a simple probem which is funny to tell and that problem was on manipulating objects where the question was to add a spouse key to terah and assign it with value of adam.

This question is very simple in the way that many people can understand but when I was working on it, my idea was too far from the level of the problem.

After spending like 15 minutes on it, I asked myself why not I'm not doing research because was not not a problem to post on discord, I took decision of googling instead of spending another time on it and I was so surprised on how the solution was just one line of code.

After resolving the question I was asking myself why I needed to go to google on simple question like this and I was laugh at myself but I learnt that programming is asking us to googling every time because the code we are not using at the moment is easy to forget it.
